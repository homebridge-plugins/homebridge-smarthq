Homebridge SmartHQ

The Homebridge SmartHQ plugin allows which allows to interact with SmartHQ API.

Installation

  1. Search for "SmartHQ" on the plugin screen of Homebridge Config UI X.
  2. Click Install.

Configuration

  1. Input Username & Password
  2. Click Save
  3. Restart Homebridge
